介绍

由于每次项目很多功能都是通用的,而我们又不需要太多功能,引用angular、react等框架并不合适,那样需要很多额外的学习时间,对新人的技术能力要求也比较高.所以我们自己做个一个包含大量常用功能,学习起来也更快捷的框架.如果您有任何建议,欢迎再github上给我们提交issue
