说明

pages，meJS分页插件，使用时需要require导入，例：

me.require(['pages'],function(){
  /*
    需要通过require导入加载pages插件。
  */
});

参数

名称 类型 可选 默认值 描述
ele 分页标签DOM：'#id 或 .class' 否 分页标签DOM节点ID或者Class。
current 
current : 1
 否 分页起始页数，实例为从第1页开始。
backFn 
backFn : function(p,isPages){
  //TODO
}
 否

分页callback函数，里面返回两个参数，分别为p,isPages：

p参数是当前显示的页码。

isPages参数，用在有查询条件的分页，表示是否重新初始化分页DOM。

代码 

/* HTML Code*/
<ul class="m_fanye"></ul>

/* JS Code */
/*
  me.pages单独使用：
*/
me.require(['pages'],function(){
  me.pages({
    ele:'.m_fanye',//分页标签节点ID或者Class
    current:1,//开始页数，从第一页。
    backFn:function(p,isPages){//点击分页的执行函数,p参数是点击的页码。
      //TODO
    }
  });
});

/*
  me.pages配合后台me.ajax使用：
*/
me.pages({
  ele:'.m_fanye',//分页标签节点ID或者Class
  current:1,//开始页数，从第一页。
  backFn:function(p,isPages){//点击分页的执行函数,p参数是点击的页码。
    isPages = isPages || 'undefined';
    me.ajax({
      key : 'products_pc',
      data : {
        current : p,//请求当前页数数据
      },
      callback : function(data){
        //TODO
      }
    });
  }
});
