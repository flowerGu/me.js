  • 说明

说明

browser，判断当前浏览器类型。

参数

名称 类型 可选 默认值 描述

输出

名称 类型 描述
browser Object 
{
  ie : '如果当前是ie浏览器则返回版本号',
  chrome : '如果当前是chrome浏览器则返回版本号',
  firefox : '如果当前是firefox浏览器则返回版本号',
  opera : '如果当前是opera浏览器则返回版本号',
  safari : '如果当前是safari浏览器则返回版本号'
}

代码 

var n = me.browser();

n.ie;
n.chrome;
n.firefox;
n.opera;
n.safari;
