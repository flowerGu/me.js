说明

replaceTpl，meJS页面模板插件，使用时需要require导入，例：

me.require(['replaceTpl'],function(){
  /*
    需要通过require导入加载replaceTpl插件。
  */
});

页面定义模板数据：

注：模板需写在script标签中，并且type类型为“text/tpl”。

<script type="text/tpl" id="tips_money_top_inner">
<ul>
    <li>平台运营<strong id="operationYear">{{operationYear}}</strong>年<strong id="operationMonth">{{operationMonth}}</strong>个月<strong id="operationDay">{{operationDay}}</strong>天</li>
    <li>累计注册人数<strong id="reg_num">{{reg_num}}</strong>人</li>
    <li>累计出借金额<strong id="deal_amount">{{deal_amount}}</strong>元</li>
    <li>累计收益额<strong id="yield_amount">{{yield_amount}}</strong>元</li>
</ul>
</script>

参数

名称 类型 可选 默认值 描述
tpl '#id' 否 模板标签对应DOM ID。
data

js对象，渲染数据：

{
  deal_amount : "84",
  operationDay : "18",
  operationMonth : "0",
  operationYear : "1",
  reg_num : 32826,
  trade_num : 15272,
  yield_amount : "558"
}
 是 模板渲染对应data数据。
callback 
callback : function(str){
  //TODO
}
 否

模板渲染callback函数，渲染完返回模板html字符串str。

代码 

/* HTML Code */
<div class="tips_money_top_inner"></div>

<script type="text/tpl" id="tips_money_top_inner">
<ul>
    <li>平台运营<strong id="operationYear">{{operationYear}}</strong>年<strong id="operationMonth">{{operationMonth}}</strong>个月<strong id="operationDay">{{operationDay}}</strong>天</li>
    <li>累计注册人数<strong id="reg_num">{{reg_num}}</strong>人</li>
    <li>累计出借金额<strong id="deal_amount">{{deal_amount}}</strong>元</li>
    <li>累计收益额<strong id="yield_amount">{{yield_amount}}</strong>元</li>
</ul>
</script>

/* JS Code */
var data = {
  deal_amount : "84",
  operationDay : "18",
  operationMonth : "0",
  operationYear : "1",
  reg_num : 32826,
  trade_num : 15272,
  yield_amount : "558"
};

me.replaceTpl({
  tpl: '#tips_money_top_inner',
  data : data,
  callback : function(str){
    $('.tips_money_top_inner').html(str);
    //TODO
  }
});
