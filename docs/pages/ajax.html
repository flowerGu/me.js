me.ajax

  • 说明

说明

ajax，向后台接口发送请求。

参数

名称 类型 可选 默认值 描述
key

通过me.config配置url字典，调用：

key : "login"

url地址调用：

key : "https://www.mejinrong.com/me-pc/login_pc.json"
 否

1、key是通过me.config配置url字典调用，请先在项目config.js里配置好url字典。

2、不需要通过me.config则直接传入后台url地址。
data

后台所需要的参数

data : {
  name : "lilin"
}
 是

1、后台需要请求参数，则通过data设置。
log 
log : 'print'
 是

1、控制台console.log输出打印内容，可用于调试。
lock 
lock : '#id 或 .class'
 是

1、锁定给定元素，防止用户连击，或是ajax回调函数没有返回，造成发送请求多次。

2、ajax返回成功或失败后，给定元素解锁。
callback 
callback : function(data){

}
 是

1、ajax请求成功，失败回调函数。

2、成功返回json数据

3、超时，请求失败返回错误字符串，"请求超时!"、"系统错误!"。

代码 

me.ajax({
  key : 'login',
  data : {
    name : 'lilin'
  },
  log : 'print',
  lock : '#button',
  callback : function(data){
    //TODO
  }
});
