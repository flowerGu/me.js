说明

me.timeout(options)

页面显示倒计时

参数

名称 类型 可选 默认值 描述
options Object 否 {}类型的参数
options.dom selector | jQuery 对象 否 jQuery对象,$('xxx')
options.text 倒计时文本 是 {{time}}秒后重新获取... 倒计时过程中显示的文本,{{time}}表示当前剩余时间
options.retry 倒计时结束文本 是 当前元素的 text()值 倒计时结束后显示的文本(比如 再发一次验证码)
options.cb 回调函数 是

输出

名称 类型 描述
(直接返回) Boolean 验证结果,通过为true,否则false
如果提供了 options,监听options.dom对象,并在input事件执行回调 options.cb

代码

//me.timeout(options)
me.timeout({
  dom:'#getVerifyCode',
  text:'{{time}}秒后重新获取',
  retryTxt:'重新获取'
});

me.timeout({
  dom:$('#test2'),
  text:'{{time}}秒后有惊喜',
  cb: function () {
    alert('倒计时按钮回调事件--倒计时完毕执行');
  }
});
