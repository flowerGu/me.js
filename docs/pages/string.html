说明

me.str.str2char(str, [start,][length,][char])

字符串部分替换(比如手机号星化 18511111111 -> 185****1111)

参数

名称 类型 可选 默认值 描述
str String 否 {}类型的参数
start Int 是 3 开始位置
length 长度 是 4 从开始位置持续的长度
char 替换的字符 是 * 将原始字符替换成的最终字符

输出

名称 类型 描述
(直接返回) Boolean 验证结果,通过为true,否则false
如果提供了 options,监听options.dom对象,并在input事件执行回调 options.cb

代码

//me.str.str2char(str, [start,][length,][char])
me.str.str2char('18514472344');//185****2344
me.str.str2char('602988068',3,4,'*');//602****68
me.str.str2char('18514472344',3,4,'-');//185----2344

说明

me.str.queryString([namr,][url])

获取查询字符串

参数

名称 类型 可选 默认值 描述
name String 是
url String 是 window.location.href 要查询的url

输出

名称 类型 描述
字符串或者对象 Boolean
如果提供name,返回该name对应的值,否则返回包含所有参数的对象

代码

//me.str.queryString([name,][url])
//以 https://google.com?name=yyj&age=25&tel= 为例
me.str.queryString();//{name:'yyj',age:'25'}
me.str.queryString('age');//'25'
me.str.queryString('tel')// ''
me.str.queryString('birthday') //null
me.str.queryString('url','https://google.com?url=baidu.com');//'baidu.com'
