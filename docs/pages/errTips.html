说明

errorTips，显示toast提示信息，使用时需要require导入，例：

me.require(['errorTips'],function(){
  /*
    需要通过require导入加载errorTips插件。
  */
});

参数

名称 类型 可选 默认值 描述
text string 否 提示消息的文本内容
time number 是 3000 提示消息展示的时间
form number 是 0 提示消息显示动画的起始位置
to number 是 80 提示消息显示动画的结束位置
width string 是 50% 提示消息的宽度
left string 是 50% 提示消息定位距离左侧的位置
callback function 是 80 提示消息消失后执行的回调函数

代码 

me.errorTips({
		text:"请输入正确的姓名!",
		time:3000,
		form:20,
		to  :150,
		width:'300px',
		left:'300px',
		callback:function(){
			//----
		}
	});
