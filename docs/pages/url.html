说明

通过url地址传参，并返回对象。

参数

名称 类型 可选 默认值 描述
url String 是 window.location，当前页面url

1、不传参数url，则处理当前页面url

2、传入要处理的url，如：http://www.baidu.com?name=lilin

输出

名称 类型 描述
url Object

http://www.baidu.com?name=lilin

{
  name : 'lilin'
}

代码 

//http://www.baidu.com?name=lilin
var n = me.url();

n.name; //lilin

var n = me.url('http://www.baidu.com?name=lilin');

n.name; //lilin
