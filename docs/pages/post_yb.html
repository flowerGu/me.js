说明

post_yb，动态创建form表单，post方式提交。

参数

名称 类型 可选 默认值 描述
url 字符串 否 通过参数url，动态创建form表单，并设置action属性为url地址。
params 
{
  name : 'lilin',
  age : 22
}
 是 通过参数params，设置提交form表单所需参数，参数格式为JS对象。

代码 

me.post_by('https://www.mejinrong.com',{
  name : 'lilin',
  age : 22
})
